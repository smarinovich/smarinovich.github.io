Keywords and Syntax

Full Guide & How to use on Help page

Note: You can not have a function of the same time inside itself (e.g. a WHILE loop in a WHILE loop). BEGIN - Start of Algorithm, must always be the first line.
END - End of Algorithm must always be the last line.
input - Allows users to input a value (all values must be a number, or a string in quotation marks).
output - Output the value to the users.
IF...THEN - First line of an if statement, the comparison must be inside the 2 words.
ELSE - Alternate Option, if IF statement gets a False results.
ENDIF - Last line of the IF statement, comes after the else section if that exists.
AND - To compare if A AND B are both true in IF statements. Will return true if both A and B are true.
OR - To compare if A OR B is true in IF statements. Will return true if A is true, B is true or Both is true.
CASEWHERE - The first line of multiway selection, followed by the value and comparisons (all cases must be a value, ending with a colon).
OTHERWISE: - Default case, which will occur if no case can, in a CASEWHERE statement.
ENDCASE - The last line of a CASEWHERE statement.
WHILE - First line of a WHILE loop, followed by a comparisons statement.
ENDWHILE - Last line in a WHILE loop.
REPEAT - Start of a REPEAT loop, which is followed by action processes in each new line.
UNTIL - The last line of the REPEAT loop, where the comparison will be.
FOR...=...TO... STEP... - First line of the FOR loop. The first gap is the variable, the second one being the first value of the for loop, next is the last value that will be used in the loop, and step is the the value that the variable will increase or decrease by.
NEXT... - The final line of a for loop, must have the same variable as in the first line.
If there is an error that can't be easily resolved, then check the help page.
